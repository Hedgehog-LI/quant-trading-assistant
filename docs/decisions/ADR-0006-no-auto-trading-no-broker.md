# ADR-0006: 不自动交易、不连接券商

- 状态：Accepted
- 日期：2026-06

## 背景
系统目标是交易辅助（记录、复盘、风控、信号），不是自动下单平台。

## 决策
不连接真实券商账户，不读取/保存券商密码、交易密码、交易 API Key；不自动下单；不输出"稳赚/必涨/无风险"结论；所有买卖提示表达为"辅助信号 + 风险提示 + 人工确认"。

## 原因
- 合规与安全：避免密钥泄露与未经授权的下单。
- 产品定位：辅助决策而非替用户决策。

## 影响
持仓快照/价格均手工维护或后续限定 provider 接入；不在此项目实现券商对接。所有"建议"文案须含免责声明。

## 替代方案
接券商 API 自动下单——明确拒绝。
