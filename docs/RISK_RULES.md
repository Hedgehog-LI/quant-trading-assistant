# Risk Rules

## 核心观点

新手亏损通常不是因为不会看指标，而是因为不会控制风险。系统中的任何买入信号都必须经过风险模块过滤。

## 必备公式

```text
单笔亏损金额 = 总资金 × 单笔风险比例
买入股数 = 单笔亏损金额 / 每股止损距离
单票最大仓位 <= 总资金 × 单票上限比例
总仓位 <= 总资金 × 当前市场环境仓位上限
```

## 默认参数建议

仅用于学习和模拟，不能视为投资建议：

| 参数 | 新手默认值 |
| --- | --- |
| 单笔风险比例 | 0.5% - 1% |
| 单票最大仓位 | 10% - 20% |
| 总仓位上限：趋势向上 | 60% - 80% |
| 总仓位上限：震荡 | 30% - 50% |
| 总仓位上限：趋势向下 | 0% - 20% |
| 连续亏损 2 次 | 降低仓位 |
| 连续亏损 3 次 | 暂停交易并复盘 |

## 做 T 风控

- 必须有底仓和机动仓区分。
- 单次 T 仓位不宜过大。
- 做 T 失败后不能无限补仓。
- 趋势向下时减少或停止做 T。
- 跌破关键结构位时优先处理风险，不要为了做 T 硬扛。

## 风险模块输出

每个信号至少输出：

- `riskLevel`: LOW / MEDIUM / HIGH
- `riskNote`: 风险说明
- `maxPositionRatio`: 最大建议仓位
- `stopLossPrice`: 止损参考价
- `invalidReason`: 如果信号被否决，说明原因

## 交易暂停机制

触发以下条件时，系统应提示暂停交易：

- 连续 3 笔亏损。
- 单日亏损超过预设阈值。
- 当前市场环境为 bearish 且个股跌破关键均线。
- 策略回测在近期样本失效。
- 用户没有写交易计划，只是临盘冲动交易。
